# 🎓 Student Marks Management System

A full-stack web application designed to manage student academic records efficiently. This project allows users to add, update, search, and delete student records while automatically calculating total marks, percentage, and grades.

Built as a learning project to understand real-world software development concepts such as REST APIs, backend architecture, database integration, CRUD operations, and frontend-backend communication.

---

## 🚀 Features

### Student Management

* Add new student records
* Edit existing student details
* Delete student records
* View all students in a dashboard
* Search students by Roll Number

### Automatic Calculations

* Total Marks Calculation
* Percentage Calculation
* Grade Generation

### Database Integration

* Store student records in MongoDB
* Retrieve and update data dynamically
* Persistent data storage

### Dashboard UI

* Responsive layout
* Modern user interface
* Easy-to-use student management panel

## ▶️ Getting Started

### Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/student-marks-management-system.git
```

### Backend Setup

```bash
cd backend
mvn spring-boot:run
```

### Frontend Setup

Open:

```text
frontend/index.html
```

in your browser.
